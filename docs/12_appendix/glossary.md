# Thuật ngữ (Glossary)

-   **Mã vận đơn (Waybill Code / Tracking Number):** Mã số duy nhất định danh gói hàng do đơn vị vận chuyển cấp.
-   **HID (Human Interface Device):** Chuẩn kết nối thiết bị ngoại vi, ở đây ám chỉ máy quét mã vạch hoạt động như bàn phím.
-   **Fragmented MP4 (fMP4):** Định dạng video MP4 được chia thành các đoạn nhỏ (fragments), giúp file không bị hỏng nếu quá trình ghi bị ngắt đột ngột.
-   **Continuous Scan:** Chế độ quét liên tục, không cần thao tác tay trên màn hình.
-   **Local-first:** Chiến lược ưu tiên lưu trữ và xử lý dữ liệu trên thiết bị người dùng thay vì server.
