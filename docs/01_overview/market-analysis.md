# Phân tích Thị trường & Đối thủ cạnh tranh

## 1. Tổng quan thị trường
Thị trường Việt Nam hiện có nhiều giải pháp hỗ trợ quay video đóng gói, nhưng đa số đi theo hướng SaaS (thu phí định kỳ) và tập trung vào mô hình Cloud.

## 2. Các đối thủ chính

| Sản phẩm | Đặc điểm | Giá | Nền tảng |
|---|---|---|---|
| **Ecombox** (ecombox.vn) | App + Desktop + Hub phần cứng, AI phát hiện lỗi, cloud storage | 3 gói (Basic/Premium/Enterprise) | iOS, Android, Win, Mac, Linux |
| **GHC** (goihangchuan.vn) | Đa nền tảng, tra cứu nhanh | Chưa công bố | Đa nền tảng |
| **ECAM** (ghihinh.com) | Desktop, 2-16 camera/bàn | 3,000,000 VNĐ/năm | Windows |
| **ECAMPMCS** | Desktop, 10-16 camera/máy | Theo gói thiết bị | Windows |
| **EcomPMS** | Web-based, giám sát nhân viên | ~9,000-10,000 VNĐ/ngày | Web |

## 3. Điểm khác biệt của Allship Record (USP)

Hầu hết đối thủ đều là SaaS trả phí hàng tháng/năm và yêu cầu cloud.

| Điểm so sánh | Đối thủ hiện tại | Allship Record (Chúng ta) |
|---|---|---|
| **Chi phí** | SaaS trả phí hàng tháng/năm | **Free local-first**, chỉ trả phí nếu dùng tính năng Cloud Sync |
| **Lưu trữ** | Bắt buộc Cloud (tốn phí, phụ thuộc internet) | **Local-first**, Cloud là optional |
| **Setup** | Cần đăng ký tài khoản, cấu hình phức tạp | **Cài app -> Dùng ngay** (không bắt buộc login) |
| **Quyền riêng tư** | Video đẩy lên server bên thứ 3 | **Video ở trên thiết bị người dùng** |
| **Thiết bị** | Thường yêu cầu/bán kèm phần cứng riêng | **Tận dụng điện thoại/PC có sẵn** (BYOD) |

## 4. Chiến lược tiếp cận
-   **Giai đoạn đầu (MVP):** Tập trung vào Seller nhỏ (< 50 đơn/ngày) sử dụng điện thoại cá nhân. Miễn phí hoàn toàn để chiếm thị phần user base.
-   **Giai đoạn sau:** Cung cấp giải pháp Desktop cho Seller lớn, bán gói Cloud Sync và các tính năng quản lý nâng cao (Enterprise).
