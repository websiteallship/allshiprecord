# Tương thích Thiết bị

## 1. Mobile (Android)
App tối ưu cho phân khúc tầm trung (Mid-range) phổ biến tại Việt Nam.

### Cấu hình tối thiểu (Minimum Specs)
-   **OS:** Android 10+
-   **RAM:** 4GB
-   **CPU:** Snapdragon 6-series hoặc tương đương
-   **Bộ nhớ trống:** 5GB
-   **Hỗ trợ:** Camera2 API Level 'LEGACY' trở lên.

### Cấu hình đề nghị (Recommended)
-   **OS:** Android 12+
-   **RAM:** 6-8GB
-   **CPU:** Snapdragon 7/8-series
-   **Bộ nhớ trống:** 20GB+
-   **Hỗ trợ:** Camera2 API Level 'FULL' hoặc 'LEVEL_3'.

### Danh sách thiết bị đã test (Verified List)
-   Samsung Galaxy A53, A54, A34, S20 FE.
-   Xiaomi Redmi Note 11, Note 12 Pro.
-   Oppo Reno 8, A78.
-   Realme C55.

## 2. Desktop (PC/Laptop)
App Electron (Windows).

### Cấu hình đề nghị
-   **OS:** Windows 10/11 (64-bit).
-   **RAM:** 8GB (Electron tốn RAM).
-   **CPU:** Core i3 Gen 8+ hoặc Ryzen 3 3000+.
-   **Ổ cứng:** SSD (để ghi video không bị drop frame).

## 3. Máy quét Barcode
Tương thích mọi máy quét hỗ trợ chuẩn **HID Keyboard**.
-   Có dây: Honeywell 1250g, Datalogic QW2100.
-   Không dây: Zebra DS2278, Netum C750.
