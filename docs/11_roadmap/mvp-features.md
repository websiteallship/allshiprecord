# Tính năng MVP (Phase 1)

Các tính năng BẮT BUỘC PHẢI CÓ để phát hành phiên bản đầu tiên.

## Core Features
1.  **Scan:** Quét được mã vạch 1D (Code 128) và QR Code qua Camera sau.
2.  **Record:** Tự động bắt đầu quy sau khi quét thành công.
3.  **Stop:** Nút dừng quay thủ công & lưu video.
4.  **Playback:** Xem lại video vừa quay.
5.  **Search:** Tìm kiếm lại video cũ bằng cách nhập mã đơn.

## Storage
1.  **Format:** File `.mp4` lưu trong thư mục công khai của App.
2.  **Naming:** Tên file chứa mã đơn.
3.  **Cleanup:** Tự động cảnh báo khi dung lượng trống < 1GB.

## Settings
1.  Tùy chọn độ phân giải (720p / 480p).
2.  Chế độ Flash (Bật/Tắt khi quay).
