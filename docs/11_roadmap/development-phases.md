# Lộ trình Phát triển (Development Roadmap)

## Phase 1: MVP (Mobile Android)
**Mục tiêu:** Ra mắt bản Android chạy ổn định cho các Shop nhỏ.
-   [x] Core: Quét mã bằng Camera.
-   [x] Core: Quay video H.264 720p lưu Local.
-   [x] Core: Tra cứu video theo mã đơn.
-   [x] Feature: Tự động xóa video cũ.
-   [ ] Release: Publish lên Google Play hoặc file APK nội bộ.

## Phase 2: Mở rộng (iOS + Hardware)
**Mục tiêu:** Hỗ trợ iPhone và các Shop chuyên nghiệp hơn.
-   [ ] Platform: Build iOS version.
-   [ ] Hardware: Tích hợp Bluetooth Scanner (HID).
-   [ ] Workflow: Continuous Scan Mode (Quét liên tục).
-   [ ] Feature: Quy trình nhận Hàng Hoàn (Return).
-   [ ] Feature: Nút xuất video/chia sẻ nhanh.

## Phase 3: Desktop Power User
**Mục tiêu:** Phục vụ Shop lớn (> 500 đơn/ngày).
-   [ ] Platform: Electron App (Windows).
-   [ ] Hardware: Hỗ trợ USB Webcam & Scanner.
-   [ ] Advanced: Multi-camera (Toàn cảnh + Chi tiết).
-   [ ] Advanced: Hỗ trợ IP Camera (RTSP).

## Phase 4: Cloud & Enterprise
**Mục tiêu:** Kiếm tiền (Monetization) & Quản lý tập trung.
-   [ ] Cloud: Sync metadata real-time.
-   [ ] Cloud: Backup video (Gói trả phí).
-   [ ] System: Phân quyền User (Nhân viên/Quản lý).
-   [ ] API: Tích hợp trực tiếp API sàn TMĐT (Shopee/Lazada) để tự động đối soát trạng thái đơn hàng.
