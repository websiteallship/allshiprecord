# Thiết lập Môi trường Phát triển (Environment Setup)

## Yêu cầu Hệ thống
-   Windows 10/11 hoặc macOS.
-   RAM: 16GB+.
-   Android Studio + SDK.
-   VS Code.

## 1. Setup Flutter (Mobile)
1.  Cài đặt Flutter SDK (Stable channel).
2.  Chạy `flutter doctor` để kiểm tra.
3.  Cắm thiết bị Android thật, bật USB Debugging.
4.  Chạy: `flutter run`.

## 2. Setup Electron (Desktop)
1.  Cài Node.js (LTS version).
2.  `npm install`.
3.  Chạy dev mode: `npm start`.

## 3. Thiết bị Test
Cần chuẩn bị ít nhất:
-   1 Điện thoại Android tầm trung (Samsung A series).
-   1 Máy quét Barcode USB (hoặc Bluetooth).
