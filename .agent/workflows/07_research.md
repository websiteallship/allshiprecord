# Nghiên cứu (Research)

## Bước 1: Xác định Phạm vi
1. Câu hỏi cụ thể cần trả lời là gì?
2. Section docs nào sẽ được cập nhật? (`docs/...`)

## Bước 2: Thu thập Thông tin
1. Tìm kiếm best practices trên web.
2. Kiểm tra tài liệu chính thức (Flutter, Dart, SQLite...).
3. Review code patterns trong các dự án tương tự.

## Bước 3: Tổng hợp
1. Viết kết quả theo format ngắn gọn.
2. Nêu rõ ưu/nhược điểm nếu so sánh các lựa chọn.

## Bước 4: Cập nhật Docs
1. Thêm kết quả vào file `docs/` phù hợp.
2. Nếu là chủ đề mới, tạo file mới trong section phù hợp.

## Bước 5: Ghi chép
1. Thêm entry vào `.ai/work_log.md`.
2. Ghi lại các câu hỏi còn mở cần nghiên cứu thêm.
